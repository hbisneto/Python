# InstaFollow Bot

O robô percorre os seguidores do usuário autenticado. O robô também segue todos os usuários percorridos, desde que o usuário autenticado não siga a conta percorrida ainda.

## GeckoDriver

Proxy para usar clientes compatíveis com W3C [WebDriver](https://developer.mozilla.org/en-US/docs/Web/WebDriver "Sobre WebDriver") para interagir com navegadores baseados no Gecko.

Este programa fornece a API HTTP descrita pelo [Protocolo WebDriver](https://w3c.github.io/webdriver/#protocol "Protocolo WebDriver") para se comunicar com os navegadores Gecko, como o Firefox. Ele traduz as chamadas para o [Protocolo Remoto Marionette](https://firefox-source-docs.mozilla.org/testing/marionette/ "Protocolo Remoto Marionette"), agindo como um proxy entre as extremidades local e remota.

> Observação: O WebDriver não está disponível nesse projeto, mas pode ser baixado diretamente do repositório do projeto. Que fica [nesse link](https://github.com/mozilla/geckodriver/releases/tag/v0.28.0)
> <br> O sistema está configurado para executar o driver que fica no mesmo diretório do arquivo 'InstaFollowRobot.py'.
> Verifique e altere o código, para executar o driver em outro diretório.

#### Recomendação de Software:

* Ambiente Virtual para manter o sistema em execução
* Mozilla Firefox
* GeckoDriver
